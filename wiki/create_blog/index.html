安装Node.js

简言

Node.JS是搭建静态博客必须配置的环境，如果配置不当将会带来未知的错误，建议仔细阅读。

Windows安装Node.js

打开Node.js官网下载博客框架所支持的版本即可(一般情况下选择长期维护版)，按照引导点击Next即可。

安装 nvm

nvm可以使您方便的管理您的Node.JS

测试安装是否成功

若没有安装成功（概率很小）请设置环境变量（电脑基础知识）或重新安装。

安装Git

Windows安装Git

Git下载与Node.JS安装过程几乎相同，一笔带过。

Linux安装Git

Ubuntu:sudo apt install git
CentOS:yum -y install git

测试安装是否成功

终端输入：git --version
